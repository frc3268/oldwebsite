Programming with Java

Overview

For programming our robots, we use Java, which is conveniently taught by the IB Computer Science. It's essentially the exact same thing as C#, for anyone who's taken that course.

If you want to learn about Java at your own pace and with high quality instruction, check out UW's CSE 142 (Comp. Sci 1) course website! It has lots of detailed examples and thorough lecture slides to learn from, and you might as well put your parents' tax dollars to good use (and, if you learn it really well, you can take the AP CS exam for some cheap college credits since the test is loosely based on UW's course).

Environment Setup (Visual Studio Code & GitHub)

  1. Download Visual Studio Code and install it on your computer!
  2. Download the JDK (version 8) appropriate for your operating system, and install it.
  3. Install the WPILib plugin software in Eclipse. OUTDATED
    • Open Eclipse
    • Click "Help" -> "Install New Software..." in the menu
    • Click the button that reads "Add..." (in the top right, next to the "Work with..." selector)
    • For "Name", enter "FRC Plugins"
    • For "Location", enter "http://first.wpi.edu/FRC/roborio/release/eclipse/"
    • Click "Add" or press enter
    • Click the arrow next to "WPILib Robot Development" in the main display on the screen to expand it, then check the box next to "Robot Java Development" that appears.
    • Click "Next" at the bottom and wait.
    • Click "Next" at the bottom again.
    • Pretend to read the license agreement, and accept the terms. William will give you a quiz later.
    • Click "Finish"
    • When the system protests, click "Install Anyways"
    • When prompted, click "Restart Now" to apply the changes.
  4. Create a GitHub account, if you don't have one.
  5. Download the GitHub Desktop client, and install it.
  6. If you'd like, activate the native git integration in Eclipse. OUTDATED
    • Open Eclipse.
    • Click "Navigate" -> "Show In" -> "Git Repositories" in the menu.
    • Position and/or resize the resulting window however you like.
  7. Give your GitHub username to Joey, then accept the invitation to join our team's GitHub organization.
  8. Clone the GitHub repository on which you are working, and follow instructions to maintain your own separate branch to avoid unnecessary merge conflicts with others.
    • GitHub Desktop
      • Click "File" -> "Clone Repository" in the menu
      • Navigate to the "URL" tab
      • Enter "frc3268/[respostory name]"
      • "Choose" your desired local path
      • Click "Clone" to finish
      • Open the newly cloned files in Eclipse (through "File" -> "Open Projects from File System..." in the menu)
    • Eclipse "Git Repositories"
      • Click "Clone a Git repository"
      • Enter "https://github.com/frc3268/[repository name].git" into the "URL" field
      • Enter your GitHub username & password in the "Authentication" section
      • Click "Next" twice
      • Open your project, which should appear in the "Project Explorer" menu
